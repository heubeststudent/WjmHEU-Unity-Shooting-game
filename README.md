## 开发

```bash
代码仓库上传项目中所用到的脚本文件以及预制体模型供开发者参考
项目支持键鼠与手柄两种操作模式
```

```bash
本项目采用Unity进行开发支持pc-windows，与安卓两种平台，开发语言采用c#，开发IDE采用VS2022
```
